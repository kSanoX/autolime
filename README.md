AutoLime is a web application for generating explanations for machine learning model predictions.

🔧 Technologies
Frontend: React
Backend: Laravel
Interface: responsive, easy to work on different devices

Functionality:

Data loading

Model selection

Visualization of the influence of features on the result

🎯 Goal
To create a tool that allows users to quickly get interpretations of ML predictions in a convenient format.

📦 Features
Clean and structured code

Simple UX

Flexible architecture for scaling
